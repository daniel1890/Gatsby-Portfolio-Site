# Gatsby Portfolio Website

## Project Informatie

Dit wordt een Portfolio website die wat informatie over mij laat zien, de pagina's die de website gaat hebben zijn:

- Index Pagina
- About Me Pagina
- Projecten Pagina

Met behulp van Gatsby is het makkelijk om een statische website te bouwen die een snelle response tijd heeft. Ook is het met Gatsby makkelijk om een CMS / Headless CMS te koppelen aan de website en om deze data dan te gebruiken/transformeren met Gatsby.
